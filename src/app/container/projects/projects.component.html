Air pollution forecasting using RNN with LSTM


Air pollution (PM2.5) is increasingly serious with industrial development. Therefore, to effectively monitor and forecast PM2.5 concentration is an important issue. This project is an approach to forecast PM2.5 concentration using RNN (Recurrent Neural Network) with LSTM (Long Short-Term Memory). exploit Keras, TensorFlow, to build a neural network and run RNN with LSTM through TensorFlow.
